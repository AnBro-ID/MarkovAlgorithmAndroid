Нормальные алгорифмы Маркова

Что это такое?

Нормальные алгорифмы (алгоритмы) Маркова (НАМ) - это один из способов формализации понятия алгоритма, предложенный советским математиком А.А. Марковым.

НАМ задает правила преобразования некоторой исходной строки при помощи списка, состоящего из строки-образца -> строки-замены.

Программа для НАМ состоит из списка пронумерованных строк. Список просматривается сверху вниз и при обнаружении слова-образца в исходной строке происходит его замена. После каждой такой замены список начинает просматриваться сначала.

Слово-образец может отсутствовать, в этом случае слово-замена ставится в самое начало исходной строки. Обычно такая замена должна стоять последней в списке во избежание зацикливания. Также может отсутствовать и слово-замена, в таком случае слово-образец удаляется из исходной строки. Символ «.» после слова-замены позволяет остановить выполнение программы.

Как работать с программой?

В дополнительном меню программы находится пункт «Условие задачи», клик по которому открывает поле редактора, где можно ввести условие задачи в свободной форме.

Поле редактирования исходной строки находится вверху окна программы.

Имеется возможность запомнить исходную строку во внутреннем буфере и восстановить обратно.

Настройки позволяют менять скорость выполнения НАМ.

В таблице в нижней части окна набирается программа. В первом столбце записаны номера строк, он заполняется автоматически. Во втором столбце записывается слово-образец, а в третьем - слово-замена. Для выделения конкретной строки достаточно нажать на её порядковый номер.

Четвертый столбец может содержать комментарий к каждой строчке программы. Добавить, удалить и переместить строки таблицы можно с помощью кнопок, расположенных справа от таблицы.

Строка, которая сейчас выполняется, подсвечивается зеленым фоном.

Возможно пошаговое выполнение программы.

Задачи для НАМ можно сохранять в файлах. Сохраняется условие задачи, программа, исходная строка. При загрузке задачи из файла и сохранении в файле исходная строка автоматически записывается в буфер. Для открытия диалогового окна «Сохранить как» для уже открытого файла нужно зажать кнопку с иконкой дискеты.
