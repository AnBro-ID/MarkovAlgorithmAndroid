Markov algorithm

What it is?

Markov algorithm - is a string rewriting system that uses grammar-like rules to operate on strings of symbols. Markov algorithms are named after the Soviet mathematician Andrey Markov, Jr.

MA sets the rules for converting some origin string using a list consisting of a pattern string -> replacement string.

The MA program consists of a list of numbered lines. The list is scanned from top to bottom, and when a pattern string is found in the origin string, it will be replaced by replacement string. After that the list will be scanned from the beginning.

The pattern string may be empty, in which case the replacement string will be placed at the very beginning of the origin string. Typically, this replacement should appear last in the list to avoid endless loop. The replacement string may also be empty, in which case the sample string will be removed from the origin string. The symbol "." placed at the end of the replacement string allows to stop the program execution.

How to work with app?

In action bar menu there is an item "Task description", it contains editor field where you can enter task description in a free form.

Edit field for origin string is placed on top of the app window.

You can save the state of origin string in the internal buffer and load it back.

Settings allow you to choose the speed of MA execution.

First column in program table contains row numbers and it is filled automatically. In the second column you can write pattern string, and in the third you can write replacement string.

Fourth column can contain a comment for selected row. You can add, delete or replace table rows, but first you need to select row by clicking on it's number.

Executing row has a green background.

It is possible to run program in step-by-step mode.

Programs for MA can be saved in files. When you load a file and save it, the initial state of origin string is automatically written to the buffer. To open "Save as" dialog box you need to hold button with the floppy disk icon.
